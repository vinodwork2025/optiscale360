Content Management Setup

Option 1: Local Studio

Run Sanity Studio locally on your computer

1. Open Terminal/Command Prompt

2. Navigate to project folder:

cd optiscale360/studio

3. Start the studio:

npm run dev

4. Open http://localhost:3333

Option 2: Deploy Studio

Deploy to Sanity's cloud for online access

1. Open Terminal/Command Prompt

2. Navigate to studio folder:

cd optiscale360/studio

3. Deploy the studio:

npm run deploy

4. Follow the prompts to choose URL

Option 3: Use Sanity.io

Access your project directly on Sanity.io

1. Go to sanity.io/manage

2. Login to your account

3. Find project: n8rucjn3

4. Click "Open Studio" or "Content"
